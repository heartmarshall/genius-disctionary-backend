## 1 - переход с RefCatalog на предзагруженные датасеты
1. Собираем датасет из поплуярных: книг, песен, сериалов и фильмов. Достаем из них слова, формируем список из 10-20 тысяч слов, которые встречюатся в них. 
2. Далее для каждого слова с помощью LLM + открытых датасетов подкачиваем: определение, транскрипцию, примеры (пока только ИИ сгенерированные, но потом нужно научиться собирать примеры из того контента, который уже загружен на шаге 1),интересные заметки про само слово - происхождение, интересные факты итд, и другие поля, которые нужны для добавления слова в Ref словарь
