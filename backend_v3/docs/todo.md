# TODO 0: 
Разные режимы повторения + добавлять карточку более точечно к смыслу слова

Direct Recall (Прямое вспоминание).
Reverse Recall (Обратное вспоминание).
Visual Association (Визуальная ассоциация).
Auditory Recall (Аудирование / Диктант).
Context Cloze (Заполнение пропуска).

Режим	Стимул	Скрыто	Роль
1. Direct	Слово	Значение	База, работает всегда
2. Reverse	Значение	Слово	Активное вспоминание
3. Visual	Картинка	Слово	Образное мышление
4. Auditory	Звук	Слово	Восприятие на слух
5. Cloze	Контекст	Слово	Грамматика и использование
6. Дают слово - нужно написать предложение с ним. Потом можно смотреть какие предложения ты писал раннее.
